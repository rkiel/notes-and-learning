# Style

## Inline

```html
<h1 style="color: red">Hello</h1>
```

## Style tag

```html
<head>
  <style>
    h1 {
      color: red;
    }
  </style>
</head>
```

## Include

```html
<head>
  <link ref="stylesheet" href="main.css" />
</head>
```

## Rules

Rule => selector(s) { declarations }

Declaration => property: value;
